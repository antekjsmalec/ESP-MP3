# ESP-MP3
A mp3 player powered by an esp32.

## Features
 - Battery powered
 - Mp3 playback
 - Volume Controll
 - Possibilty for simple apps and games

## Parts List
 - Esp32 development board
 - 0.96" OLED screen
 - PCM 5102A digital to analog converter
 - SD card module
 - SD card (No bigger than 32gb)
 - Buttons
 - Wires
 - Li-Po Battery
 - USB-C Battery charging board
 - Simple on/off switch (Optional)
